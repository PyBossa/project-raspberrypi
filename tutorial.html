Tutorial

Hi! The goal of this project is to find out if outside a window in a house -located in Madrid, Spain- there is a hive of wasps or bees.

The issue is that, this house is 5th floor, so it's not easy to get out and see if there is a hive or not. Thererfore, we decided to use the Raspberry Pi PyBossa integration to find it out, taking a photo every 5 minutes.

The project is really simple. It loads a photo taken using a Raspberry Pi camera in that window and asks you this question: Do you see an insect in this photo?

You will have 5 possible answers:

  • No,
  • Yes! A Wasp
  • Yes! A Bee
  • No photo
  • I don't know

All you have to do is to click in one of the possible answers and you will be done. The project has always new photos to analyze, as the Raspberry Pi takes a picture every 5 minutes. Thus, if you like the project, come back whenever you want!
